## Key Classes and Methods

### KAPScraper
```python
scraper = KAPScraper(delay=1.5, timeout=30, max_retries=3)

# Company management (cached)
companies = scraper.get_company_list()  # → list[dict]

# Financial reports
disclosures = scraper.get_financial_disclosures(
    ticker="THYAO",
    year=2024,
    period=None,
    limit=10
)  # → list[FinancialDisclosure]

# Detailed financial tables
tables = scraper.get_financial_tables(disclosure_id)  # → FinancialTables

# Material events
specials = scraper.get_special_disclosures(
    ticker="THYAO",
    start_date="2024-01-01",
    end_date="2024-12-31",
    limit=50
)  # → list[SpecialDisclosure]
```

### Data Models (Pydantic)
```python
FinancialDisclosure(

SpecialDisclosure(

    summary: str | None,
    full_text: str | None,

FinancialTables(
    disclosure_id: str,
    ticker: str,
    period: str,
```

## Implementation Highlights

### Rate Limiting
- Configurable delay between requests (default 1.5s)
- Prevents IP blocking from KAP
- Automatically applied to all requests

### Retry Logic
- Exponential backoff with urllib3.Retry
- Handles transient 5xx errors
- Configurable max retries (default 3)

### Fallback Support
- Hardcoded company list (8 major BIST stocks)
- Graceful degradation when API is unavailable
- Logging for monitoring fallback usage

### Request Caching
- Company list cached after first fetch
- Avoids redundant API calls
- Configurable per-instance

### Data Persistence
- Raw JSON saved to `data/raw/{category}/` for debugging
- Facilitates replay and analysis
- Automatic directory creation

### Type Safety
- Full type hints throughout
- Pydantic validation for data integrity
- IDE autocompletion support
