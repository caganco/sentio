# DEC-006: Kelly Criterion Position Sizing

**Decision Date:** 10 May 2026  
**Status:** 💡 PENDING (Phase 5 blocker)

---

## CONTEXT

Currently, positions are **fixed-size** (each stock gets same allocation). But signal strength varies:
- Strong signal (0.85 score) → should size 5%
- Weak signal (0.45 score) → should size 1%
- Neutral signal (0.50 score) → should size 0%

---

## DECISION (PENDING APPROVAL)

Implement Kelly Criterion position sizing:

**Formula:**
```
f* = (win_rate - loss_rate/odds) / odds

Position Size = f* × account_size (capped 0.5%-5%)

Constraints:
• Portfolio max 10% drawdown (Kelly fraction × position count)
• Minimum 0.5% size (avoid micro-positions)
```

**Example:**
```
Signal Score: 0.75 (strong buy)
• Win probability: 65%
• Kelly f*: (0.65 - 0.35/2) / 2 = 0.2 (20%)
• Capped: 5% max
• Position Size: 5%

Signal Score: 0.50 (neutral)
• Win probability: 48%
• Kelly f*: -0.04 (negative = skip)
• Position Size: 0% or 0.5% minimum
```
