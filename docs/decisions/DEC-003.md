# DEC-003: Macro-Equity Correlation Layer

**Decision Date:** 12 May 2026  
**Status:** ✅ IMPLEMENTED

---

## CONTEXT

Layer 2 (Macro Intelligence) provides macro signals: USD/TRY, Brent Oil, CDS, yield curve. But missing: **how does each stock respond to macro changes?**

**Problem:** 
- USD/TRY ↑ → Exporters benefit, importers hurt
- Brent Oil ↑ → Energy stocks down, consumers up
- Generic macro signal doesn't account for sector sensitivity

**Solution:** Build correlation matrix per stock to macro factors, then weight macro signals by sensitivity.

---

## DECISION

Add Layer 2.1: Macro-Equity Correlation Matrix

**Calculation:**
```
Adjusted Macro Signal = (USD/TRY signal) × (GARAN sensitivity to USD/TRY)
                       + (Brent signal) × (GARAN sensitivity to Brent)
                       + ...

Sensitivity = rolling 60-day correlation to actual returns
```

**Result:** Personalized macro signal per stock (not one-size-fits-all)

## IMPLEMENTATION

**File: `src/signals/layer2_macro.py`**

```python
class MacroEquityCorrelation:
    def calculate_stock_sensitivity(self, ticker, lookback_days=60):
        """
        Compute 60-day rolling correlation:
        ticker_returns → macro_changes
        """
        returns = self.get_returns(ticker, lookback_days)
        
        sensitivities = {
            'usd_try': pearsonr(returns, self.macro_data['usd_try_changes'])[0],
            'brent': pearsonr(returns, self.macro_data['brent_changes'])[0],
            'cds': pearsonr(returns, self.macro_data['cds_changes'])[0],
            'yield_curve': pearsonr(returns, self.macro_data['yield_curve_changes'])[0]
        }
        
        return sensitivities
    
    def adjust_macro_signal(self, ticker, macro_signals, sensitivities):
        """Weight macro signals by stock sensitivity."""
        adjusted = (
            macro_signals['usd_try'] * sensitivities['usd_try'] +
            macro_signals['brent'] * sensitivities['brent'] +
            macro_signals['cds'] * sensitivities['cds'] +
            macro_signals['yield_curve'] * sensitivities['yield_curve']
        ) / sum(abs(v) for v in sensitivities.values())
        
        return adjusted
```
