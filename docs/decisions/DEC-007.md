# DEC-007: Ruthless Alpha Philosophy — Remove Defensive Constraints

**Decision Date:** 16 May 2026  
**Decided By:** Strategist (the maintainer)  
**Status:** ✅ DECIDED

---

## CONTEXT

The current signal engine applies conservative averaging and defensive constraints that reduce conviction in edge cases:

1. **Composite formula weights everything equally:** `tech*0.20 + macro*0.35 + risk*0.05 + kap*0.15 + smart*0.25 + sent*0.00`
   - Spreads conviction across 5 layers
   - A strong 95/100 technical signal is diluted by weaker macro data
   - Each layer gets a "voice" regardless of signal strength or consistency

2. **Hard thresholds (BUY-STRONG @ 72, BUY-WEAK @ 60):** Uniform boundaries ignore layer confidence
   - A 72 signal from 5 weak inputs ≠ a 72 signal from 2-3 very strong inputs
   - No reward for conviction concentration (multiple layers aligned)

3. **Risk layer always active:** `risk*0.05` penalizes even when technical setup is pristine
   - VIX spike or USDTRY move can trigger risk-off on profitable trade setup
   - Defensive posture conflicts with Alpha objectives

4. **Sentiment deactivated:** `sent*0.00` (awaiting DistilBERT Phase 4.2.1)
   - Dead weight in formula
   - Frees 5% allocation

**Problem:** System is calibrated for stability, not Alpha. Multiple weak signals average out to "meh" (60–70 range) when we should be saying "NO EDGE" or "STRONG NO."

---

## OPTIONS CONSIDERED

### Option A: Keep Current Weights (Status Quo)
- Composite formula unchanged: `tech*0.20 + macro*0.35 + …`
- Hard thresholds stay: 72 = BUY-STRONG, 60 = BUY-WEAK
- **Pros:** Stable, predictable, regression-free
- **Cons:** No Alpha upside; defensive averaging dilutes conviction
- **Cons:** Ignores correlation between layers (all signals aligned = stronger edge)

### Option B: Conviction Scoring (CHOSEN)
- Replace composite average with **conviction-based scoring:**
  - Count how many layers are aligned (same signal direction)
  - Weight by layer importance (macro > technical > risk)
  - Boost signal when multiple strong sources agree
  - Suppress signal when layers conflict
- Dynamic thresholds based on conviction count:
  - 1 layer: -15 pts from base (weak, noisy)
  - 2 layers: base threshold
  - 3+ layers: +15 pts (strong conviction, take larger position)
- **Pros:** Rewards edge conviction, penalizes conflicting signals
- **Pros:** Aligns with Alpha objectives (concentrate capital where edge is clearest)
- **Cons:** More complex logic, requires testing Phase 4.2.3

### Option C: Layer Dominance (Hard Cutoff)
- If macro signal > 75, use macro only (ignore others)
- If technical signal > 85, take technical signal with 50% position size
- If risk score < 30, veto all buy signals (hard stop-loss)
- **Pros:** Simple if-then rules
- **Cons:** Loses information; too binary
- **Cons:** Creates dead zones in signal space

### Option D: Ensemble Voting (Democratic)
- Each layer votes BUY/HOLD/SELL
- Signal = majority vote + weighted score
- **Pros:** Intuitive
- **Cons:** Ignores magnitude (95/100 tech = same vote as 51/100 tech)

---

## DECISION: Option B (Conviction Scoring)

**Rationale:**

1. **Alpha requires conviction:** In edge cases (high VIX, tight spreads, tail risk), only the signals most certain should trigger trades
2. **Layer alignment is information:** When technical, macro, and smart money all point the same direction, that edge is real. When they conflict, noise dominates.
3. **Defensive averaging fails:** Spreading 1.0 conviction across 5 layers = 0.2 conviction each. That's not a trade.
4. **Risk layer is a filter, not a weight:** Risk signals should gate entry (veto buy if risk > threshold), not dilute the signal
5. **Sentiment is waiting:** 5% currently wasted; reallocate to conviction boost when DistilBERT Phase 4.2.1 lands

```python
def compute_signal_ruthless_alpha(self, ticker):
    """
    Conviction-based signal computation.
    Rewards alignment, penalizes conflict.
    """
    scores = self._compute_layer_scores(ticker)
    
    # Step 1: Get layer signals (direction: BUY/HOLD/SELL)
    tech_signal = self._classify_layer(scores['technical'], 'technical')
    macro_signal = self._classify_layer(scores['macro'], 'macro')
    kap_signal = self._classify_layer(scores['kap'], 'kap')
    risk_signal = self._classify_layer(scores['risk'], 'risk')
    
    # Step 2: Risk gate (hard stop)
    if risk_signal == FinalSignal.SELL_STRONG:
        logger.info(f"{ticker}: Risk veto triggered (score < 30)")
        return SignalResult(final=FinalSignal.HOLD, conviction=0, reason="risk_gate")
    
    # Step 3: Conviction count (alignment)
    conviction_layers = self._count_aligned_layers(
        [tech_signal, macro_signal, kap_signal]
    )  # Skip risk in count (it's a gate, not a signal)
    
    # Step 4: Composite base (weighted average of aligned layers only)
    composite_base = self._weighted_composite(scores, conviction_layers)
    
    # Step 5: Dynamic threshold adjustment
    threshold_adjustment = {
        1: -15,  # Single layer: weak signal
        2:   0,  # Two layers: base threshold
        3:  +15, # Three layers aligned: strong conviction
    }.get(conviction_layers, 0)
    
    # Step 6: Apply thresholds with conviction boost
    thresholds = {
        'buy_strong':  72 + threshold_adjustment,
        'buy_weak':    60 + threshold_adjustment,
        'hold_upper':  60 + threshold_adjustment,
        'hold_lower':  48 + threshold_adjustment,
        'sell_weak':   32 + threshold_adjustment,
    }
    
    final_signal = self._classify_composite(composite_base, thresholds)
    
    return SignalResult(
        final=final_signal,
        conviction=conviction_layers,
        composite_base=composite_base,
        threshold_adjustment=threshold_adjustment,
        layer_signals={
            'tech': tech_signal,
            'macro': macro_signal,
            'kap': kap_signal,
            'risk': risk_signal,
        }
    )

def _count_aligned_layers(self, layer_signals):
    """Count how many layers agree on direction (BUY/SELL)."""
    buys = sum(1 for s in layer_signals if s in [FinalSignal.BUY_STRONG, FinalSignal.BUY_WEAK])
    sells = sum(1 for s in layer_signals if s in [FinalSignal.SELL_WEAK, FinalSignal.SELL_STRONG])
    holds = sum(1 for s in layer_signals if s == FinalSignal.HOLD)
    
    # Alignment = max(buys, sells, holds) / total
    max_agree = max(buys, sells, holds)
    return max_agree  # 1, 2, or 3
```

## EXPECTED OUTCOMES

### Before (Defensive Averaging)
```
Scenario: EURUSD weakness + KAP earnings + Technical breakout
- Technical: 85/100 (BUY-STRONG)
- Macro: 70/100 (BUY-WEAK)
- KAP: 65/100 (BUY-WEAK)
- Risk: 50/100 (HOLD)

Composite: (85*0.20 + 70*0.35 + 50*0.05 + 65*0.15 + ...) / 1.0 = ~67
Signal: BUY-WEAK (threshold 60)

Result: Lukewarm confidence, smaller position size
```

### After (Ruthless Alpha)
```
Same scenario:

Composite base: (85*0.20 + 70*0.35 + 65*0.15) / 0.70 = ~73
Conviction layers: 3 (all aligned on BUY)
Threshold adjustment: +15 (three layers agree)
Adjusted threshold: 72 + 15 = 87

Signal: BUY-WEAK (73 < 87, but > 60)

Result: Conviction reward, larger position size (but still respects threshold)
```

### Defensive Alpha (Risk Gate Works)
```
Same setup BUT risk_score = 25 (below 30 gate):

Composite: 73 (same calculation)
Risk Gate: TRIGGERED (score < 30)

Signal: HOLD (hard veto)

Result: Alpha protection; noisy risk signal stops bad trade
```
